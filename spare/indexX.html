Utilisation de Font Awesome sans connexion Internet

Pour utiliser les icônes de Font Awesome en local, donc sans accès à Internet, il suffit d’inclure le fichier fontawesome-all.min.js dans le projet. Il peut être téléchargé préalablement ici : https://git.io/vAWvg.

À noter que dans la version 3 de Bootstrap, les icônes proposées étaient celles de Glyphicons. Dans la version 4, ils proposent d’autres options et Font Awesome n’en fait pas partie.
